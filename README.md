# Average, Sum & Difference Calculator (C#)

This C# console application demonstrates how to:

- Use methods with `out` parameters  
- Return multiple values from a function  
- Perform basic arithmetic operations  
- Print results to the console  

## 📌 What the Program Does

The program:

1. Calls a method named `Average`
2. Calculates:
   - The average of three numbers  
   - The sum of three numbers  
   - The difference of three numbers  
3. Displays all results on the console
